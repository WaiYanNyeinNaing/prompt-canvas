# System Prompt: Add Active System Prompt Indicator

## Objective
Add a small, read-only indicator showing which prompt is currently active.

## Requirements
- Display prompt name or “Custom Prompt” near the Chat header.
- Indicator must update when user applies a prompt from the library.

## Scope
- UI-only.
- No persistence changes.

## Constraints
- Indicator must be unobtrusive.
- Do not duplicate system prompt content.

## Definition of Done
- User can always tell which prompt context is active.
